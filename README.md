# 💖 Compliment Machine

A single-page, no-dependencies compliment generator. Click a button, get complimented.
Gets progressively more unhinged the higher you turn the dial.

🔗 **Live site:** https://rmanabat-eng.github.io/compliment-machine/

## Features

- **Compliment Intensity slider (1–10)** — ranges from professional LinkedIn-style
  praise up to full Shakespearean-sonnet drama.
- **Compliment Style picker** — choose between Default, 📻 Boomer, or 💀 Gen Z voice.
- **Name input** — personalize compliments with your name.
- **Compliment history panel** — see everything you've received this session.
- **5 color themes** — pink, purple, blue, yellow, green.
- **Confetti + bounce animation** on every click.

## Usage

Just open `index.html` in any browser — no build step, no dependencies.

Or visit the live version: https://rmanabat-eng.github.io/compliment-machine/

## Tech

Plain HTML, CSS, and JavaScript in a single file. No frameworks, no npm install.
